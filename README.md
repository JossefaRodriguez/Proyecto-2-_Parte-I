# 📊 Proyecto 2 – Parte I: Análisis Exploratorio de Múltiples Datasets

Este repositorio contiene el análisis exploratorio de cuatro datasets distintos, con el objetivo de identificar patrones, valores atípicos y preparar los datos para modelado predictivo. Cada notebook incluye diagnóstico, visualizaciones, selección de problema y propuestas de solución.

---

## 📁 Datasets incluidos

| Dataset        | Descripción breve | Problema seleccionado | Notebook |
|----------------|-------------------|------------------------|----------|
| 🏗️ Concreto     | Predicción de resistencia del concreto según su composición | Regresión | `EDA_dataset1_Concrete.ipynb` |
| 🏥 Seguros médicos | Análisis de costos médicos según perfil del paciente | Regresión | `EDA_dataset2_insurance_Medical_Cost_Per.ipynb` |
| 🐟 Peces         | Estimación del peso de peces a partir de medidas físicas | Regresión | `EDA_dataset3_Fish.ipynb` |
| 🏠 Real Estate   | Predicción del precio por unidad de superficie de propiedades | Regresión | `EDA_dataset4_realstate_precioVentaPropie.ipynb` |

---

## 📊 Metodología común

Cada notebook incluye:

- ✅ Diagnóstico inicial del dataset
- 📉 Visualización de distribuciones y outliers (boxplots, histogramas)
- 🔥 Mapa de calor de correlaciones
- 🎯 Selección de problema (regresión supervisada)
- 🧪 Propuesta de solución y preparación para modelado
